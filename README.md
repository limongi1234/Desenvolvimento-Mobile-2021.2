# Desenvolvimento Mobile (2021.2) 📱

Trabalhos da disciplina de **Desenvolvimento Mobile** (semestre 2021.2), desenvolvidos em **React Native** com **Expo**. O repositório reúne dois projetos práticos de interface mobile.

## 📂 Projetos

### 🎯 Desafio
Aplicativo construído com **Expo**, explorando o sistema de layout do React Native (componentes `View`, Flexbox, posicionamento e estilização de telas).

### 💬 Clone (WhatsApp)
Recriação da **interface do WhatsApp** em React Native, com componentes reutilizáveis (`Header`, `Icones`) e organização da UI em `src/components`.

## 🚀 Como executar

```bash
git clone https://github.com/limongi1234/Desenvolvimento-Mobile-2021.2.git

# entre na pasta do projeto desejado (Desafio ou Clone)
cd Desenvolvimento-Mobile-2021.2/Desafio

npm install
npx expo start
```

Abra no **Expo Go** (Android/iOS) lendo o QR Code, ou rode no emulador.
